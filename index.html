HTML

  • Adding 'alt' information to images assists screenreaders.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element is used for things like displaying copyright information.
  • Headers are ordered h1-h6 from largest to smallest and help with readability of the page.

Git

  • Git checkout will move between branches, -b flag will create a new branch.
  • It is always important to make sure to pull the main branch before commiting changes to ensure that the information you have is up to date.
  • Git status: Checks the branch we are currently in.
  • Never adjust code directly in the main branch, always pull to a feature or development branch beforehand.
